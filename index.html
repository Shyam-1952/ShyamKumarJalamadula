E-COMMERCE DEVOPS Implementation on Microsoft Azure (AKS + IaC + CI/CD)

Type: Individual Project

Tools Used: Microsoft Azure, Azure Kubernetes Service (AKS), Terraform, Docker, Docker Compose, Azure DNS, Argo CD, Azure Load Balancer, Kubernetes (Manifests, Services, Volumes)

Description:
This project showcases a complete end-to-end DevOps implementation on a containerized E-Commerce application using Microsoft Azure. It includes setting up Azure resources like Azure AD roles, Azure VMs, NSGs, and VNets, followed by Docker, Docker Compose, and Kubernetes setup. Infrastructure provisioning is done using Terraform with proper backend and state management. The application is deployed on Azure Kubernetes Service (AKS) using Kubernetes manifests with LoadBalancer services and storage configurations. A custom domain is mapped via Azure DNS, and GitOps is implemented using Argo CD.
