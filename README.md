# Shyam Kumar Jalamadula | DevOps Portfolio

## Description
A personal portfolio website for a DevOps Engineer, showcasing professional summary, technical skills, professional experience, projects, personal projects, education, and additional information.

## Key Features
*   Responsive design for various screen sizes.
*   Glassmorphism UI effect for sections with hover animations (scale, blur, glow effect).
*   Dynamic hero animations using GSAP.
*   Scroll animations using AOS.
*   Justified text for professional summary.
*   Bold contact information with removed email underline.
*   Visually distinct project cards with borders.

## Technologies Used
*   HTML5
*   CSS3 (with Glassmorphism, responsive design)
*   JavaScript (GSAP for animations, AOS for scroll effects)
*   Font Awesome (for icons)

## Setup and Usage
1.  Clone the repository: `git clone [repository-url]`
2.  Navigate to the project directory: `cd my-resume`
3.  Open `index.html` in your preferred web browser.
